Custom Media Recommender App - AI image generation and audio suggestions tool with responsive UI.

🚀 Features

AI-powered image/text generation

Audio content discovery

Responsive design

Get personalized podcast recommendations

Simple and responsive user interface

Real-time data fetched from a podcast API

Easy to use search functionality

🛠️ Technologies Used

HTML – Structure of the webpage

CSS – Styling and layout

JavaScript – Logic and API integration

Podcast API – Fetches podcast data based on user input

⚙️ How It Works

The user enters a topic in the search bar.

JavaScript sends a request to the podcast API.

The API returns a list of podcasts related to the topic.

The application displays the podcast results on the webpage.
